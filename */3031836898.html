2018/06/05 00:00 凌晨
目录

转载原文评论: 阿里云 ecs 上安装了 mysql 但是外网访问不了

1.阿里云 ECS 有安全组防火墙.默认只开了 ICMP,22,3389. 在服务器控制台找到 安全组-配置规则-添加安全组规则

2.如果是用 root 登录 MySQL,在登陆后 use mysql, user 表 root 对应的 localhost 改为%

3.mysql 默认只能自本机访问

4.检查是不是在监听状态

5.防火墙是否对 mysql 端口限制,设置防火墙
