基本使用

1. 注册 OAuth Application

New OAuth ，填写内容时候确保 callback URL 与你的博客地址域名一致。

2.引入 js、css 到文章页面

3. 初始化评论

页面发布后，你需要访问页面并使用你的 GitHub 账号登录（请确保你的账号是第二步所填 repo 的 owner），点击初始化按钮。
